# Classic LED Blink 💡

This repository contains my very first steps into the world of embedded systems. Even writing this read-me part makes me feel very excited about the future :) See you in better and more detailed projects in sha Allah.

## Hardware Configuration
* Microcontroller: Arduino Uno 
* Component: 1x Standard LED (or 2x)
* Resistor: 220 Ω (Ideal resistance value for safe operation)
* Connection: Digital Pin 8
